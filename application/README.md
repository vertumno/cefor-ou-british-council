# application/ — Workspace de redação da candidatura

Aqui ficam os **textos que vão para o formulário** de submissão (Good Grants). Cada arquivo corresponde a um bloco do Sample Application Form. Escreva em **inglês** (exigência do edital); mantenha um resumo PT-BR quando útil para validação interna.

| # | Arquivo | Bloco do formulário | Responsável sugerido |

## Convenção de status no topo de cada arquivo

```
> STATUS: rascunho | em revisão | pronto para o formulário
> ÚLTIMA EDIÇÃO: AAAA-MM-DD por <nome>
```

## Princípios não-negociáveis (sob pena de gate de eliminação)

1. **Não é pesquisa.** Verbos de exploração/co-design, nunca "estudar o impacto" ou "rodar o MOOC".
2. **Sem PI/licença no orçamento** (inelegível) — vai como contribuição in-kind.
3. **Co-design paritário visível** — contribuição intelectual do CEFOR explícita (ADDIEM, expertise MOOC, inclusão).
4. **Gênero é gate** — foco demonstrável em mulheres/meninas, com indicadores.
5. **ODA + ODS** — ligar a desenvolvimento e a pelo menos um ODS.
