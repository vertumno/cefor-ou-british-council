# Direção recomendada (Caminho 1, reposicionado)

> **Co-desenhar um framework de equivalência de crédito e reconhecimento entre uma microcredencial inclusiva existente da OU (FHEQ Nível 7) e a certificação MOOC do IFES, mapeando as barreiras regulatórias brasileiras, com um piloto inclusivo voltado a professoras da Rede Federal.**

Índice de aprovação estimado: **88/100**. Ver comparação em [[proposal-path-options]].

## Por que esta formulação vence (mapa para os critérios)

| Movimento | Gate/critério blindado |
|-----------|------------------------|
| Ancorar numa microcredencial **já existente** (HZVM884/885) | Foge da armadilha pesquisa/produção; mandato exato da [[partners-perryman]] |
| Núcleo = **mapeamento regulatório + framework de reconhecimento** | Maior peso dos 35%; impacto de sistema, não pesquisa ([[brazil-regulatory-recognition]]) |
| **ADDIEM** (Vanessa) co-aplicado + co-autoria | Co-design paritário (Quality + Resources) ([[partners-battestin]]) |
| Piloto com **professoras** + herança do MOOC premiado de inclusão | Gate de gênero/EDI + ambiente TNE inclusivo |
| **Sem IP no orçamento**; workshops/missões/frameworks | Caps + custos elegíveis ([[grant-budget-and-costs]]) |
| Continuidade via CONIF/CONCEFOR + trilha do MA Online Teaching + MoU | Sustainability 25% (post-grant pathway) |
| Felipe como camada de acessibilidade + ponte; IA só como "IA responsável" | Evita leitura de projeto de IA/pesquisa ([[partners-tessarolo]]) |

## Pacotes de trabalho sugeridos (Fase 2, 17 meses)

1. **Governança paritária** (meses 1–2): comitê diretor conjunto, papéis, plano de trabalho.
2. **Mapeamento de reconhecimento** (meses 2–6): barreiras regulatórias (Carolina Bori, vácuo de microcredenciais, carga horária MEC) + proposta de equivalência via OUVP/RPL.
3. **Co-adaptação da microcredencial** (meses 4–10): sprints conjuntos; ADDIEM + learning design do IET; acessibilidade (WCAG, VLibras) embutida.
4. **Piloto inclusivo** (meses 9–14): coorte de **professoras** da Rede Federal; métricas de alcance e inclusão.
5. **Consolidação** (até jun/2028): disseminação no CONCEFOR, MoU, roteiro para edital maior, relatório final.

## Indicadores (Methodology, 30%)

- Nº de barreiras regulatórias mapeadas e validadas com stakeholders.
- Framework de equivalência de crédito produzido e revisado pelas duas instituições (entregável tangível).
- Microcredencial co-adaptada (blueprint) + nº de participantes no piloto, desagregado por gênero.
- MoU assinado + roteiro de continuidade (post-grant pathway).

## Os 3 outros caminhos como pacotes

- **Caminho 2** (capacitação/comunidade de prática) → frente de disseminação e capacity building.
- **Caminho 3** (diálogo de política/QA) → pacote de sistema; evolução pós-grant.
- **Caminho 4** (microcredencial conjunta co-certificada) → fase seguinte, depois do reconhecimento mapeado.
